# Contributing to CANVAS

> Welcome! This project welcomes contributions to improve the Credit Union Agent Network.

## How to Contribute

### Reporting Issues
- Check existing issues first
- Use clear titles and descriptions
- Include steps to reproduce if applicable

### Suggesting Enhancements
- Open an issue with the `enhancement` label
- Describe the proposed change
- Explain why it would benefit credit union operations

### Pull Requests

1. **Fork** the repo
2. **Create** a feature branch (`git checkout -b feature/my-feature`)
3. **Commit** your changes (`git commit -am 'Add new feature'`)
4. **Push** to the branch (`git push origin feature/my-feature`)
5. **Open** a Pull Request

## Guidelines

### Agent Definitions (in `agents/`)
- Follow the existing markdown format with `name`, `description`, `color` frontmatter
- Include clear role descriptions and core responsibilities
- Add critical rules and deliverables sections

### Strategy Documents (in `strategy/`)
- Maintain consistent heading structure
- Keep cross-references accurate
- Update the file structure in EXECUTIVE-BRIEF.md if adding new files

### Workflows
- Document compliance gates clearly
- Include quality gates and handoff points
- Add escalation protocols where appropriate

## Coding Standards

- Use clear, descriptive file names
- Include frontmatter for agent definitions
- Keep markdown consistent with existing files
